Welcome

This web app is designed specifically for Meghalaya Government employees to provide a quick and efficient way to compute their salary and other pay-related calculations. Built with an understanding of the unique pay structure and allowances applicable to government employees, this tool ensures accuracy and simplicity in your calculations. \n\nPlease note that this app is an independent initiative and is not sponsored or endorsed by any agency or client. It is created solely to assist you in streamlining your pay computations with ease and convenience.

Project Overview

This web application was developed to help government employees quickly calculate their Pay, DA Arrear and Transfer TA amoong others simply by entering their basic pay, increment percentage, number of months, and other necessary information. This is inspired by the 5th Pay Commission of Meghalaya, which governs the salary structure for government employees.

Technologies Used

The web application is built using basic web technologies such as:

  • HTML for the structure and content
  • CSS for styling and responsiveness
  • JavaScript for handling calculations and user interactions
